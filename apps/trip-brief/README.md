# Trip Brief (Philbin app)

A lightweight one-page trip/field brief generator:
- Optional weather snapshot (Open-Meteo; no API key)
- Sunrise/sunset (via Open-Meteo daily variables)
- Checklist templates + editable checklist
- Notes
- Export an all-day `.ics` calendar event

## Run locally

Open `apps/trip-brief/index.html` in a browser.
